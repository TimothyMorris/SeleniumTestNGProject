Set up Instructions 
1) Download the SeleniumTestNGProject and save it anywhere
2) Launch a command prompt and go in to the 'SeleniumTestNGProject' folder
3) Execute 'mvn install -DskipTests'
4) Import the project in to Eclipse as an Existing Maven Project
5) Install the TestNG Eclipse plug-in
      Go to Help -> Install New Software
      Enter http://beust.com/eclipse in to the 'Work with:' text field
6) Download the TestNG plug-in and restart Eclipse
7) Locate AmazonTestNG.xml (which lives in src/main/resources/testng)
8) Right click on it and select Run As -> TestNG Suite
9) Watch the magic unfold!
